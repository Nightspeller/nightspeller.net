BioSpecimen Registration app

My team and I developed this complicated single page web application for registration of biospecimens. It is a system with numerous interactions with user, including not only UI validation and registration of new biological specimens, but also search, editing and exploration of existing records. Application provides different possibilities for users with different roles, like admins, curators and regular scientists.

Tech: HTML5, CSS3, JS, Backbone+Marionette, Stylus, Subversion, Node.js, Brunch, Datatables, D3, other js libraries (jQuery, ect.).
